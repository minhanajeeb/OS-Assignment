# Operating System Assignment

## Program:
SRTF (Shortest Remaining Time First) CPU Scheduling

## Description:
This program implements the Shortest Remaining Time First (SRTF) CPU scheduling algorithm. It calculates waiting time and turnaround time for each process.

## Input Format:
n
PID AT BT

## Output:
- Waiting Time
- Turnaround Time
- Average Waiting Time
- Average Turnaround Time

## Language Used:
C Programming
